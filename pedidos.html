Configuración o servidor

  1. En Vercel → tu proyecto → Storage → creá o conectá Postgres (suele ser Neon, con capa gratuita).
  2. En el SQL Editor de esa base, ejecutá vercel-postgres-schema.sql. Si la tabla ya existía, también vercel-postgres-migration-fecha-pedido.sql.
  3. Hacé un nuevo deploy (push a GitHub o “Redeploy”). La API vive en /api/pedidos; no hace falta ningún archivo de claves en el front.
